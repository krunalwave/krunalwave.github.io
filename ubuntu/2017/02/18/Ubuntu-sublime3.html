Posted by Shujin Wu on February 18, 2017

Ansible Role

Run Ansible playbook: Ubuntu Environment Set Up

Packages

[Ctrl+Shift+P] -> Install Packages

  • SidebarEnhancement
  • SublimeLinter
  • SublimeLinter-pep8
  • SublimeCodeIntel
  • SublimeHaskell
  • GitGutter
  • JSLint
  • Markdown-editing (update line_numbers to true for each user setting)
  • Markdown Preview
  • Trailing-space
  • Color-highlighter
  • bracketHighlighter
  • CTag
  • ansible
  • afterglow-theme

Add Chinese Support

Enable sougou-pinyin in SublimeText 3

  • Go to /opt/sublime_text
  • Run sudo gcc -shared -o libsublime-imfix.so sublime_imfix.c pkg-config –libs –cflags gtk+-2.0 -fPIC
