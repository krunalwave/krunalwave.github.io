修复 mp3 文件导入 iTunes 时的乱码问题

Posted by Shujin Wu on July 10, 2017

TLDR:

pip install mutagen
mid3iconv -e gbk *.mp3 # fix files in the same dir

# fix files in the nested dir
find . -iname "*.mp3" -execdir mid3iconv -e gbk {} \;

Reference: 如何解决Linux下mp3乱码
